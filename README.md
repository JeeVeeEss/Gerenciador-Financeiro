# Gerenciador-Financeiro
Projeto dedicado a desenvolver um gerenciador financeiro simples. 
---
## Como funciona ?
> O usuário poderá entrar em sua conta e registrar gastos e ganhos diários, semanais e mensais com o intuito de observar sua saúde financeira.
---
 # Como testar o sistemas e fazer suas features ?
 - Clonar o repositório
 - Iniciar a Virutalenv e instalar as dependências com o Poetry
 - Usar o comando `poetry run manage.py runserver`
 - Se divertir!
